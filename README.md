# MinIo-LS
## Тест нагрузки на сервер:  
Были произведены подключения в lable studio с 7 различных устройств, не находящихся в одной сети с сервером. На каждом устройстве было открыто по 4 окна, в каждом из которых бул запущен отдельный проект.  
Свободная оперативная память до начала теста: ~8800 МБ
